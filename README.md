# giffgaff 中文使用教程

这是一个面向中文用户的 giffgaff 使用教程，目标是把激活、保号、漫游和常见故障写成清晰、可直接照着操作的图文流程。

- 购买或拿到 SIM 后如何激活
- 在中国大陆或英国境外如何保号
- 漫游、短信验证码、语音信箱和余额查询
- iPhone 上的 Wi-Fi Calling 检查
- 常见故障排查

## 最短流程

1. 打开 <https://www.giffgaff.com/activate>。
2. 输入 SIM 卡包装上的激活码。
3. 注册或登录 giffgaff 账号。
4. 按页面提示充值或购买 plan。
5. 插卡开机，等待自动注册网络；如无法上网，检查 APN。
6. 每隔一段时间进行一次有效使用或充值，避免号码长期闲置。
