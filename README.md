## 最短流程

1. 打开 <https://www.giffgaff.com/activate>。
2. 输入 SIM 卡包装上的激活码。
3. 注册或登录 giffgaff 账号。
4. 按页面提示充值或购买 plan。
5. 插卡开机，等待自动注册网络；如无法上网，检查 APN。
6. 每隔一段时间进行一次有效使用或充值，避免号码长期闲置。
